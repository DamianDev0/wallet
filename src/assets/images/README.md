# Imágenes necesarias para Auth

Por favor, agrega las siguientes imágenes en esta carpeta:

1. **welcome.png** - Imagen para la pantalla de bienvenida (similar a la imagen de "Social Chatter Team" en el diseño)
2. **login.png** - Imagen ilustrativa para la pantalla de login (personajes naranjas jugando)
3. **signup.png** - Imagen ilustrativa para la pantalla de sign up (personajes naranjas con raqueta de tenis)

## Dimensiones recomendadas
- Ancho: 300-400px
- Alto: 200-300px
- Formato: PNG con fondo transparente

## Ubicación
Coloca las imágenes directamente en `/src/assets/images/`
